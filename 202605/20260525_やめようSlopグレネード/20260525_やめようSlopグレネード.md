# やめようSlopグレネード

:::note
以前の記事「AIが書いてAIが読む時代に、人間が書く意味ってなんだろう」と関連する話題です。
今回は、AIの出力をそのまま人に渡してしまうコミュニケーションについて考えます。
:::

## 「slopグレネード」という言葉

海外のエンジニアコミュニティで「Slop grenade（スロップグレネード）」という言葉が
使われ始めています。  
"slop"は粗悪なもの・雑なもの、という意味の英語で、AIの出力を無加工のまま他者に投げる行為を指します。  
AI Slopという言葉を耳にしたことがある人もいるかもしれませんが、その延長線上にある振る舞いを指す表現です。  

チャットやメールで思い当たる場面はないでしょうか。  

- 相談したら、明らかにAIが生成した長文が返ってきた
- 聞きたかったのはその人の意見だったのに、情報の羅列だった

:::source
No Slop Grenade
<https://noslopgrenade.com/>
:::

## なぜ問題になるか

誰かに質問するとき、多くの場合はその人の判断や経験を聞きたいはずです。  
AIが整理した情報を求めているなら、自分でAIに聞けばいいのですから。  

単に「AIに聞いてみました」だけでは、情報を横流ししているだけです。  
言い換えると、自分の判断を省いたままAIの出力に依存してしまっている状態です。  

特に仕事では、相手が知りたいのは「情報そのもの」より、「あなたはどう見ているか」「どう判断したか」です。  
AIの出力をそのまま渡すだけでは、その部分が抜け落ちます。  

## 送る前のひと呼吸

AIの利用が許可されている場面において、AIを下書きや情報整理に使うのはいいでしょう。  
AIを使い始めてすぐのころは善意で全文送りたくなってしまうこともあると思います。  
ただ、その情報を相手に送る前に一点だけ確認してみてください。  

**「この返答に、自分の考えや判断が一言でも入っているか？」**  

入っていなければ、送信内容を見直した方がよいサインです。  

:::tip
AIの出力を読んで「なるほど」と思えたなら、その「なるほど」を自分の言葉にしてください。
AIに呑まれるのではなく、AIを活用して自分の思考を補助させましょう。
:::
